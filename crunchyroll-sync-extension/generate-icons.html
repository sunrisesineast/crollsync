Crunchyroll Sync Extension Icons

Quick Fix for Extension Loading Error

Click the "Generate All Icons" button below to create the required icon files automatically.

This will create simple orange gradient icons with sync arrows.

16x16 Icon


48x48 Icon


128x128 Icon


After Generating Icons:

  1. Save all three downloaded files to the icons/ folder in your extension directory
  2. Go to chrome://extensions/
  3. Click "Load unpacked" and select your extension folder
  4. The extension should now load without errors!
